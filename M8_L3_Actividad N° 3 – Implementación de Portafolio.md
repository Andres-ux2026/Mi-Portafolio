# Módulo: Portafolio Profesional — Actividad N° 3

## Instrucción 1: Construcción o actualización del portafolio

Ya contaba con un portafolio previo, por lo que se realizó una revisión y mejora completa aplicando buenas prácticas vistas en clase.

### Herramientas utilizadas

| Herramienta | Uso |
|---|---|
| **Git / GitHub** | Control de versiones y despliegue del portafolio vía GitHub Pages. |
| **GitHub Pages** | Hosting del sitio estático desde la rama `main` raíz. |
| **HTML5 UP (Solid State)** | Template base para el diseño visual. |
| **Visual Studio Code** | Editor de código durante todo el desarrollo. |
| **EmailJS** | Servicio de envío de correos desde el formulario de contacto sin backend. |
| **Google Fonts** | Tipografías Raleway y Source Sans Pro. |
| **Font Awesome** | Iconos vectoriales en el banner y sección de contacto. |

### Cambios aplicados en base a buenas prácticas

1. **Diseño responsive** — Se reemplazó un diseño anterior (Halcyonic) por el template **Solid State**, completamente responsivo, adaptándose a móvil, tablet y escritorio.
2. **Rendimiento y accesibilidad**:
   - Imágenes en WebP con fallback a PNG.
   - Imágenes con `object-fit: cover` y alt descriptivo.
   - Fuentes cargadas desde CDN con `preconnect`.
3. **SEO y Open Graph** — Meta tags para descripción, título, Open Graph y favicon.
4. **Despliegue continuo** — Publicación automática desde `main` a GitHub Pages cada vez que se hace push.
5. **Formulario funcional** — Integración con EmailJS para recibir mensajes sin backend propio.
6. **CV descargable** — Enlace directo al archivo `cv.pdf`.

### Nuevas secciones y elementos añadidos

- **Sección Sobre mí** con fotografía de perfil y texto justificado con la identidad profesional.
- **Sección de Proyectos** con 5 tarjetas (Akela Gym, Tienda Ecommerce Bootcamp, CryptoSprout, ChefReserve API, AuraFit App), cada una con descripción, tecnologías y enlaces a código y demo.
- **Sección de Habilidades** con badges de tecnologías actuales: Python, Django, JavaScript, Docker, Git, GitHub, PostgreSQL, Tailwind CSS, OpenCode.
- **Sección de Contacto** con formulario EmailJS y enlaces a LinkedIn, GitHub y correo.
- **Cita motivacional** de Steve Jobs en el footer.
- **Frase de identidad** en el header: "El buen software no se escribe, se diseña resolviendo problemas".

## Instrucción 2: Evidencia del proceso

### Herramientas utilizadas

Las mismas detalladas en el punto anterior. El proceso completo se realizó mediante el asistente OpenCode, que ejecutó comandos git, editó archivos HTML/CSS/JS, y gestionó el versionamiento directamente desde la terminal.

### Cambios aplicados

| Aspecto | Antes | Después |
|---|---|---|
| Diseño | Halcyonic (HTML5 UP) | Solid State (HTML5 UP) |
| Paleta de colores | Grises cálidos (#D4D9DD, #3B4346) | Fondo oscuro (#2e3141) con acento azul (#4c5c96) |
| Tipografía | Yanone Kaffeesatz + Inter | Raleway + Source Sans Pro |
| Cards de proyectos | Grid con Tailwind | Features flex con altura uniforme |
| Imagen de perfil | Círculo (recortada) | Rectángulo con bordes redondeados (completa) |
| Contenido | Se corrigió: primera card mostraba "EcoTrack Dash" cuando debía ser "Akela Gym" | Restaurado al original |

### Nuevas secciones añadidas

- Banner con icono circular personalizado.
- Sección Sobre mí con perfil profesional detallado.
- Cita de Steve Jobs en el footer.
- Frase de cabecera personalizada.

## Instrucción 3: Reflexión personal

**¿Cómo crees que este portafolio contribuirá a tu inserción y proyección en la industria TI?**

Este portafolio me permite mostrar de forma concreta y ordenada mi evolución como desarrollador, uniendo mi experiencia previa en administración y finanzas con las competencias técnicas adquiridas. Al estar publicado en GitHub Pages y versionado con Git, demuestra dominio de herramientas fundamentales del ecosistema TI. Las secciones de proyectos y habilidades le dan al reclutador una visión clara de lo que soy capaz de construir, mientras que el formulario de contacto facilita la comunicación directa. Además, el diseño responsivo y cuidado refleja profesionalismo y atención al detalle, cualidades que buscan los equipos de desarrollo. A futuro, este portafolio será el punto de partida para mostrar nuevos proyectos y especializaciones, proyectando una imagen de crecimiento continuo en la industria.
